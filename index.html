Free Services for Developers

A curated list of free services and tools for developers.

Hosting Services

  • DanBot Hosting - Free Discord bot hosting, Uptime Kuma hosting, Nginx (PHP) hosting, and many more services for developers.
  • GitHub Codespaces - Cloud development environment integrated with GitHub.
  • Render - Cloud application hosting with free tier for web services, static sites, and databases.
  • Neon - Free serverless PostgreSQL hosting with up to 1 databases.
  • GitHub Pages - Free hosting for static websites directly from your GitHub repository.
  • Cloudflare Pages - Fast, secure hosting for frontend developers with free tier.

Domain Services

  • is-a.dev - Free subdomain service offering name.is-a.dev domains for developers.
  • thedev.me - Free subdomain service offering name.thedev.me domains for developers.
  • DuckDNS - Free dynamic DNS hosted on AWS.
  • Afraid.org FreeDNS - Free Subdomains and dynamic DNS service.

Additional Free Services

  • MongoDB Atlas - Fully managed cloud database service with a free tier.
  • GitHub - Version control and collaboration platform with free public and private repositories.

This list is maintained by me. If you know of any free services for developers that should be added, please contribute!
